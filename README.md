# Nuria Portfolio

## Build

```bash
npm install
npm run build
npm run serve
```

This compiles the JSX sources into browser-ready files in `dist/` and keeps Babel out of the page runtime.

## Preview

`npm run serve` starts a tiny static server on `http://localhost:4173` and serves `Nuria Portfolio.html` from the project root.

## Notes

- `Nuria Portfolio.html` loads the compiled files from `dist/`.
- The generated files are not meant to be edited by hand.
